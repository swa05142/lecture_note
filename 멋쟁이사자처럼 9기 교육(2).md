#### 웹에 대한 이해와 코딩을 위한 필수 요소

#### HTML/CSS Javascript 소개

HTML = Hyper Text Markup Language 의 약자

Hyper Text = 목차를 누르면 원하는 페이지로 가지는 텍스트를 Hyper Text 라고함.

Mark up Language= 태그를 이용하여 문서나 데이터의 구조를 명시하는 언어

CSS = Cascading Style Sheets  웹 페이지에 관한 다양한 스타일 들을 정의

Style Sheets : 웹페이지의 스타일과 관련된 모든 것을 정해둠

Javascript : 웹을 이용하는 유저와 상호작용 하기 위한 기능을 추가할 때 쓰는 언어

### 제 8강 : HTML 기초

#### HTML 문서 구조

```html
<!DOCTYPE html> <!--문서 형식을 정의-->
<html lang="kr"><!--본격적인 태그의 시작, 사용하는 주 언어를 정의-->

        <p>저는 현재 웹 프로그래밍을 공부하는 학생입니다.</p>
    </body>
</html>
```

### 제 9강 : 텍스트와 관련된 태그

콘텐츠에서 큰 비중을 차지하는 것은 텍스트임.

대표적인 텍스트와 관련된 태그: 제목 태그, 본문태그, 글자와 관련된 태그



```html

```



#### 글자와 관련된 태그



```html

<sup> = 위에 기입한
```

### 제 10강 : 링크 태그

#### a 태그 

anchor의 약자

atrributes(속성)이 무조건 들어감

시작태그의 괄호 사이에 들어가고 a 한칸 띄우고 들어감.

속성 : 태그에 대해 추가적인 정보 제공

HTML의 모든 태그는 속성을 가질 수 있음

링크 주소가 들어가 있어야함.

```html
<a 링크 주소 이름 = '주소'> </a>
```



#### href

hypertext reference의 약자

연결할 웹사이트 주소를 담고 있음



#### 경로

주소 + 경로 = URL(Uniform Resource Locator)

인터넷에서 HTML페이지, CSS 문서, 이미지 등 자원의 위치를 나타냄



#### URL

절대 URL 

+ 접근하는 최초 시작점부터 경유한 경로를 모두 기록하여 리소스의 위치를 나타냄

상대 URL

+ 기준점을 기준으로 상대적인 경로를 기록하여 리소스의 위치를 나타냄

![image-20210416160948634](C:\Users\lenovo\AppData\Roaming\Typora\typora-user-images\image-20210416160948634.png)

#### target

링크를 클릭했을때 해당페이지를 어디에 열지 정하는 속성

ex) 새 탭에서 링크열기, 새 창에서 링크열기, 시크릿 창에서 링크 열기

target="_self"

+ 현재 탭에서 링크를 여는 속성

target ="_blank"

+ 새 탭에서 링크를 여는 속성

### 제 11강 : 멀티미디어와 관련된 태그

```html
<img> = 이미지 태그 <img src="이미지 url" atl = "사진 설명">
```

### 제 12강 테이블과 리스트

#### 테이블의 구성

일정한 형식이 존재함. 

```html
<table> : 표 전체를 감싸는 태그 
<td> : 표의 행 내부에 데이터 셀을 담는 태그
<th> : 표의 행 내부에 제목 셀을 담는 태그
<tr> : 표에서 행을 구분하는 태그
```



#### 테이블의 구성

rowspan = "숫자"

숫자 만큼 셀이 행을 점령하게 됨

colspan = "숫자"

숫자 만큼 셀이 열을 점령하게됨



#### 목록

순서 없는 목록

```html
<ul>
<li>아이템1</li>
<li>아이템2</li>
<li>아이템3</li>
</ul>
```



순서 있는 목록

```html
<ol>
<li>아이템1</li>
<li>아이템2</li>
<li>아이템3</li>
</ol>
```

### 제 13 강 : 폼 태그

#### action (post): 데이터를 보낼 URL을 지정

Post = URL 끝에 붙이지 않고 메세지의 바디 부분에 숨겨서 보냄

method = "post"

서버에 있는 데이터를 쓰거나 수정, 삭제할 때 주로 사용

이러한 수정과 삭제가 주소에 나타난다면 보안에 취약



#### method (get): 보내는 방식을 지정

Get = 브라우저에서 폼에 입력한 데이터를 URL 끝에 붙여 눈에 보이게 보냄 ex. 엽서

method = "get"

데이터 조회 만을 목적으로 할 때 주로 쓰임



#### input 태그

사용자에게 입력을 받기 위해 쓰이는 태그

```html
<input type = "text">  :ㅌ <input>태그의 종류를 결정
<input type = "text" name = "id"> : 태그 중 같은 타입과 구분되는 이름을 결정
```



#### placeholder

placeholder = "아이디를 입력하세요." : input에 아무 값도 입력 되지 않았을 때 나타나는 텍스트



#### value

value = "soonho" : 실제 할당되는 값, 우리가 데이터를 넣으면 이 속성에 값이 들어감. 초기값처럼 둘 수 있음



#### label

해당하는 라벨을 클릭 시 <input>태그가 활성화 됨.



#### div

태그들을 구분 짓고 나누기 위해 사용되는 태그



#### select

무조건 name태그를 갖고 있어야함. input의 네임태그와 동일함. value라는 속성을 가져야함.

```html
<select>
        <option>남성</option>
        <option>여성</option>
</select>
```
